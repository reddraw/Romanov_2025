This program has 2 main functions.

1. Hash

User inputs the data he wants to hash and program gives the hashcode.

Example:
In:Hello, world!
Out:Hash is: 315f5bdb76d078c43b8ac0064e4a0164612b1fce77c869345bfc94c75894edd3

2. Exit

The program shuts down.

Installation.

1. Make sure your Python version is 3.9 and hashlib library is installed.
2. Install repository from github
3. Open .py file in any Python compiler.
4. Press F5.
