This program has 3 main functions.

1. Cipher

User inputs the word and the key and program gives the ciphered version.

Example:
In(Word):IDIOCY OFTEN LOOKS LIKE INTELLIGENCE
In(Key):wheatson
Out:KFFBBZFMWASPNVCFDUKDAGCEWPQDPNBSNE 

2. Test
The program tests its all main parts which are needed to cipher.

In:2
Out:
Test 1 passed
Test 2 passed
Test 3 passed
Tests passed: 3/3

3. Exit
User writes 3 or any other number and program breaks.

Example:
In:3
Out:"Goodbye"


Installation.

1. Make sure your Python version is 3.9.
2. Install repository from github
3. Open .py file in any Python compiler.
4. Press F5.
